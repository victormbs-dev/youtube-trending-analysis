# YouTube Trending Videos Analysis

![YouTube Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/b/b8/YouTube_Logo_2017.svg/2560px-YouTube_Logo_2017.svg.png)

Análisis de datos de videos trending de YouTube en diferentes regiones del mundo.

## Descripción del Proyecto

Este proyecto analiza datos de videos que aparecieron en la sección "Trending" de YouTube en diferentes países. El objetivo es descubrir patrones, tendencias y relaciones entre diversas métricas para entender qué factores contribuyen a que un video se vuelva trending.

## Objetivos del Análisis

1. Consolidar datasets de múltiples regiones
2. Analizar el tiempo que tarda un video en volverse trending
3. Estudiar patrones temporales (días de la semana, fechas)
4. Analizar el uso y frecuencia de tags
5. Comparar categorías entre regiones
6. Calcular métricas de engagement
7. Investigar correlaciones entre diferentes variables
